## Features

- [x] Shell Commands
- [x] Authentication
- [x] Popping MessageBox
- [x] File I/O
- [ ] Run on startup
- [ ] Self-Update
- [ ] Screenshot
- [ ] Network Proxy

## Building

### Debug

```sh
make
```

### Release

```sh
make release
```

### Release With CNC Host

```sh
make release_host cnc_host="address.of.cnc"
```
